# 搜索项目

**github**支持高级搜索
**在readme中搜索**

```
git 最好 学习 资料 in:readme
```

![image-20210917231128690](img/image-20210917231128690.png)

这样子就会在readme文件中搜索 `git 最好 学习 资料`相关的项目

**搜索star大于某个值的项目**

```
git 最好 学习 资料 in:readme stars:>1000
```
